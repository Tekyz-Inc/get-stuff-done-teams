# Domain: commands

## Responsibility
Create all 7 new backlog slash command files following existing GSD-T command patterns.

## Owned Files
- `commands/gsd-t-backlog-add.md` — Capture item, auto-categorize, append to bottom
- `commands/gsd-t-backlog-list.md` — Filtered, ordered view
- `commands/gsd-t-backlog-move.md` — Reorder by position
- `commands/gsd-t-backlog-promote.md` — Refine → classify → create GSD-T artifact
- `commands/gsd-t-backlog-edit.md` — Modify entry fields
- `commands/gsd-t-backlog-remove.md` — Drop item with optional reason
- `commands/gsd-t-backlog-settings.md` — Manage types, apps, categories, defaults

## NOT Owned (do not modify)
- All existing command files (gsd-t-*.md, branch.md, checkin.md, Claude-md.md)
- Templates (templates/)
- CLI installer (bin/gsd-t.js)
- README.md, CLAUDE.md, CLAUDE-global template
- gsd-t-init.md, gsd-t-status.md, gsd-t-help.md (owned by integration domain)
