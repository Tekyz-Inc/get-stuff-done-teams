# Domain: templates

## Responsibility
Create template files for backlog.md and backlog-settings.md that define the file format specification used by all backlog commands.

## Owned Files
- `templates/backlog.md` — Template for .gsd-t/backlog.md with entry format and structure
- `templates/backlog-settings.md` — Template for .gsd-t/backlog-settings.md with default types, apps, categories, and defaults

## NOT Owned (do not modify)
- All existing templates (CLAUDE-global.md, CLAUDE-project.md, requirements.md, architecture.md, workflows.md, infrastructure.md, progress.md)
- Command files (commands/)
- CLI installer (bin/gsd-t.js)
