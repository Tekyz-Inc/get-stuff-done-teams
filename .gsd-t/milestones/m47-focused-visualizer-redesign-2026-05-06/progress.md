**M47 Focused Visualizer Redesign** — VERIFIED 2026-05-06.

**Goal**: Replace the current "click-to-view single spawn" visualizer model with a dual-pane focused view: top pane always streams the main in-session conversation (zero-click default), bottom pane streams the selected spawn. Left rail surfaces a Live + Completed section split so users can review any spawn's transcript long after it finished.

**Success criteria** (measurable):
1. Opening `/transcripts` with an active in-session conversation shows it streaming in the top pane within 3s of page load, **zero clicks**.
2. Clicking any rail entry loads it in the bottom pane within 1s; sessionStorage preserves selection across reload.
3. When a spawn transitions running → completed, it moves Live → Completed section without user action; if selected, stays selected.
4. Completed section displays at least 100 historical spawns capped, sorted newest-first, with status badges; toggle collapses/expands.
